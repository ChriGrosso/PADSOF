README - SkyManager

Inicio de la aplicación
------------------------

Para iniciar correctamente la aplicación, basta con ejecutar la clase:

    interfaz.prueba

Esta representa el punto de entrada principal del programa.

Credenciales iniciales
-----------------------

Si no existen datos previamente guardados, el sistema generará automáticamente un usuario gestor con las siguientes credenciales:

    NIF: 01020304A
    Contraseña: password123

Acceso a funcionalidades
-------------------------

Tras iniciar sesión como gestor, estarán disponibles todas las funciones que se pueden encontrar en la documentación.

Notas sobre la persistencia de datos
-------------------------------------

Los datos se guardan automáticamente al cerrar la aplicación.
Para restaurar el estado inicial del sistema, se pueden eliminar manualmente los archivos .dat generados.

Notas sobre las Notificaciones
-------------------------------------
i) Si se quiere aceptar la solicitud de compartir un vuelo con otra terminal, se debe pulsar leer en la notificación, de forma que saldrá un mensaje preguntando
si se quiere aceptar o no.

ii) Para establecer una hora alternativa a un vuelo de las enviadas por el gestor, se debe pulsar leer en la notificación. 
Tras esto saldrá un mensaje donde habrá que seleccionar la hora a la que se quiera trasladar el vuelo.
